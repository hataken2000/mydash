⚡ MyDash マニュアル

ブックマーク・アプリ・タスク・メモを一画面で管理するパーソナルダッシュボード。
セットとアラームを組み合わせれば、会議前に必要な資料をまとめて自動展開できる。

MyDash とは

シングルHTMLファイルで動作するローカルダッシュボード。ブラウザで開くだけで使え、データはブラウザの localStorage に保存される。ローカルサーバー（mydash-server.js）と組み合わせることで Macアプリの起動やカレンダー取得も可能。

HTMLファイル
インストール不要。ブラウザで開くだけで動く
ローカル保存
データはブラウザの localStorage に自動保存
ローカルサーバー
port 3737 でアプリ起動・カレンダー取得を中継
Gist同期
GitHub Gistを使って複数端末でデータを同期

セットアップ

必要なもの

必須Google Chrome（または Chromium 系ブラウザ）— mydash.html を開くブラウザ
必須Node.js（v18 以上推奨）— ローカルサーバー（mydash-server.js）の実行に必要
任意npm・yarn 等のパッケージマネージャーは不要（追加モジュールなし）
💡 Node.js のインストール確認：ターミナルで node -v を実行。バージョンが表示されれば OK。未インストールの場合は nodejs.org からダウンロード。

ファイル構成（~/Desktop/mydash/）：

mydash/
  ├── mydash.html    ← メインファイル（ブラウザで開く）
  ├── mydash-server.js  ← ローカルサーバー（アプリ起動・カレンダー用）
  ├── MyDash.app     ← ワンクリック起動アプリ
  └── widget.html     ← 時計＋タスク表示のウィジェット

MyDash.app で起動（推奨）

1
MyDash.app をダブルクリック

サーバーの起動と mydash.html のブラウザ表示を同時に行う

2
設定を開いて「ローカルサーバー」欄の ● が緑になれば OK

「サーバー確認中…」→「サーバー稼働中」に変わる。アプリ起動・カレンダー機能が使える状態

⚠️ サーバーなしでも基本機能（ブックマーク・タスク・メモ・セット）は使える。Macアプリ起動・カレンダー連携はサーバーが必要。

カテゴリのリネーム・削除・色変更

カテゴリ名横の をクリックするとリネームモードに入る。

リネーム

テキスト入力に切り替わり、Enter で確定・Esc でキャンセル。全アイテムのカテゴリが一括変更される

🎨 色変更

フォルダアイコンをクリックするとカラーパレットが表示される。選択した色がフォルダアイコンとノードマップの島の枠色に反映される

削除

リネームモード中の × ボタンをクリック。確認ダイアログの後、カテゴリと全アイテムのカテゴリ設定を削除

アイテムの種類

ブックマーク
URLをお気に入り登録。クリックで開く。favicon自動取得。
ツール/アプリ
MacアプリをDock名で登録。サーバー経由でワンクリック起動。URLも設定可。
タスク
ステータス（未着手/進行中/完了）・優先度・締切日時を管理。時刻も設定可能。
メモ
Markdown対応のメモ。画像・Google Mapsカード・住所の地図埋め込みに自動変換。
その他
上記に当てはまらない汎用アイテム。
セット
複数アイテムを一括起動。会議資料・朝のルーティンなどに。アラームと組み合わせて自動起動も可能。
Slack送信
Slackの特定チャンネルに定型文を素早く送信。Chrome拡張あり：入力欄に自動入力。なし（スマホ等）：クリップボードにコピーしてSlackを開く。
場所
住所・Google MapsのURLを登録。クリックでMapsを開く。説明欄に住所を書くと地図を自動表示。
タブセット
MyDash拡張機能と連携して保存済みのタブセットを復元。Chrome拡張が必要。

メモのインライン編集（リスト表示）

リスト表示でメモカードを展開し、本文エリアをクリックするとその場で直接編集できる。⌘+Enter またはエリア外クリックで保存。

アイテムの非表示

セット専用など通常は見せたくないアイテムを非表示にできる。選択モードで対象アイテムを選び「非表示にする」ボタンを押す。非表示アイテムは通常時は一覧に出ない。

非表示アイテムを再表示するには、選択モードに入ると薄く表示されるので選択して「表示にする」を押す。

ホバーボタン（リスト表示）

リスト表示でカードにホバーすると、編集・削除・複製などのアイコンボタンが右端に表示される。は左端に表示される。

選択モード

ツールバーの選択ボタンで選択モードに入ると複数アイテムを一括操作できる。選択後のアクションは「表示にする」「非表示にする」「削除」の3種類。

狭幅モード（リスト表示）

ウィンドウを狭くしてメインエリアが約450px以下になると、リストカードは自動的にコンパクト表示に切り替わる。

表示モード

ツールバーのビューボタンで表示モードを切り替えられる。選択した表示モードは自動保存される。

モード内容
⊞ グリッドカードをグリッド状に表示。展開ボタン（▼）で詳細を確認できる
≡ リスト縦一列のコンパクト表示。ホバーで操作ボタンが出る
⊡ アイコンアイコンのみを並べたコンパクト表示。大量アイテムの一覧に
◉ ノードマップカテゴリを「島」として表示し、その中にアイテムをノードで配置する俯瞰ビュー。ドラッグでパン、ホイール/ピンチでズーム。クリックで各アイテムのパネルを開ける
💡 ノードマップでは各アイテムに種類アイコンと設定済みのアイテムアイコン（ファビコン・絵文字）が表示される。カテゴリの色はサイドバーのフォルダアイコン色と連動する。

クイック追加

ツールバー左端のトグルボタンで入力モードを切り替えると、テキストを入力して Enter するだけでアイテムを即登録できる。

ボタンモードEnter の動作

メモ追加入力テキストをメモとして即登録
タスク追加入力テキストを今日締切のタスクとして即登録
💡 ボタンをクリックするたびに → → → と循環。⌘K を押すと常に検索モードに戻ってフォーカス。登録後は入力欄がクリアされ連続入力できる。

並び替え・カテゴリ変更

並び替え（ドラッグ&ドロップ）

カードをドラッグして別のカードの上にドロップすると順番を変えられる。ツールバーのソート選択で「手動」にすると並び替えが有効になる。

ソートオプション

オプション内容
手動ドラッグ&ドロップで自由に並び替え
新しい順 / 古い順登録日時で並び替え
A→Z / Z→Aタイトル順（日本語対応）
種類順アイテムの種類でグループ化
カテゴリ順カテゴリ別にグループ化して表示
優先度順お気に入り → 優先度（高→中→低→なし） → 期限日の順。タスク管理に最適

カテゴリ変更（サイドバーへドロップ）

カードをドラッグして、サイドバーの カテゴリ名 にドロップするとそのカテゴリに変更される。○ 未設定 にドロップするとカテゴリをクリアできる。

💡 カテゴリは編集ダイアログからも変更できる。入力欄は既存カテゴリのサジェスト付き。

アラーム

アラームはセットの中に設定する機能。指定した時刻にセットを自動起動できる。繰り返しスケジュールに対応。

アラームの設定（セット編集から）

1
セットを作成 or 既存セットを編集

「＋ 追加」→ 種類「セット」、または既存セットの ボタン

2
フォーム下部「アラーム」のトグルをONにする

繰り返し・時刻の設定欄が展開する

3
繰り返しと時刻を設定して保存

毎日・毎週（曜日指定）・隔週・毎月・第N曜日・毎年 から選択

4
設定時刻になるとポップアップが表示される

トースト通知とビープ音で知らせる。ポップアップから「今すぐ起動」でセットを実行できる。

スケジュール設定内容
毎日毎日同じ時刻に発火
毎週曜日を複数選択可能（月〜日）
隔週基準日から1週おきに発火
毎月（日付）毎月指定した日付に発火
毎月（第N曜日）「第2月曜」など細かく指定可能
毎年毎年指定した月日に発火

アラームの確認・管理

アラームを設定したセットはカードに 毎週 月・水… のようにスケジュールが表示される。

「アラーム管理」クリック

サイドバー上部の「アラーム管理」またはバッジをクリックすると、セット一覧が「アラームあり」でフィルタされる。

選択モードで一括ON/OFF

セットを選択モードで複数選択すると「ON」「🔕 OFF」ボタンが表示される。

💡 セット削除するとアラームも一緒に削除される。削除確認ダイアログにその旨が表示される。

集中モード（アラーム一時停止）

サイドバー上部の アイコン（アラーム管理の左）をクリックすると、全アラームを一時無効にできる。もう一度クリックで解除。

アラームON

通常状態。設定した時刻に発火する。

🔕 集中モード中

アラームが発火しない。赤みがかった表示になる。

セット

登録済みのアイテム（ブックマーク・ツール/アプリ）を複数まとめて一括起動する機能。

セットの作成

1
「＋ 追加」→ 種類「セット」を選択
2
タイトルを入力し、起動するアイテムにチェック

登録済みのブックマーク・ツール/アプリが選択肢に出る

3
アラームを設定（任意）

フォーム下部のアラームトグルをONにすると繰り返し・時刻を設定できる

4

パネル内の「起動」ボタンで一括実行。URLはChromeの新ウィンドウでタブグループとして開く。アプリはサーバー経由で起動。

💡 URLのみのセットはサーバーなしでも動作する（Chrome拡張経由で新ウィンドウ＋タブグループとして起動）。Macアプリ起動はローカルサーバーが必要。

Slack送信

Slackの特定チャンネルに定型文を素早く送れる機能。環境によって動作が異なる。

💻 PC（Chrome拡張あり）

Chrome拡張（MyDash Slack Sender）がSlackの入力欄にメッセージを自動入力する。送信ボタンはユーザーが押す。

📱 スマホ・拡張なし

メッセージをクリップボードにコピーしてSlackのURLを開く。Slackアプリに切り替えて貼り付けて送信する。

アイテムの作成

1
「＋ 追加」→ 種類「Slack送信」を選択
2
タイトルを入力して保存

チャンネルURLとメッセージはクイックパネルから設定・編集できる

3
カードをクリック → Slackクイックパネルが開く

チャンネルを選択してメッセージを入力し「送信」ボタンを押す

Chrome拡張のセットアップ（初回のみ）

1
Chromeで chrome://extensions を開く

右上の「デベロッパーモード」をONにする

2
「パッケージ化されていない拡張機能を読み込む」→ slack-extension フォルダを選択

MyDashのフォルダ内にある slack-extension/ を選択する

3
拡張機能が追加されたらセットアップ完了

MyDashをリロードすると拡張機能が認識される

タスク・メモアイテムをタップ/クリックすると、フル編集ダイアログではなくコンパクトなクイック編集パネルが開く。よく使う項目だけを素早く編集できる。

メモ

タスク

Googleカレンダー連携

データ管理

JSON エクスポート / インポート

操作場所内容

GitHub Gist 同期（複数端末）

GitHub GistをバックアップストレージとしてPC・スマホ間でデータを同期できる。

1
GitHubでPersonal Access Token（repo または gist スコープ）を発行
2
設定（）→「☁️ Gist同期」→ TokenとGist IDを入力して保存

Gist IDは空欄で保存すると新規作成される

3
「今すぐ同期」で手動アップロード。以降は起動時に自動で同期

PC側でアップロード → スマホ側でリロードするとダウンロードされる

💡 起動時に壊れた参照（削除済みアイテムへの参照がセットやアラームに残っている場合）を自動検出・修復する。修復があった場合はトーストで件数を通知する。

ウィジェット

時計と今日・期限超過のタスクだけを表示する小ウィンドウ。MyDashを操作しながら常にタスクを確認できる。

起動方法

サイドバー上部の ウィジェット をクリック → 別ウィンドウで起動する。

🕐 時計

現在時刻と日付をリアルタイム表示。

🔴 期限超過

未完了で期限が過ぎたタスクを表示。

📅 今日締切

今日が期限の未完了タスクを表示。

💡 タスクをクリックすると完了/未完了をトグルできる。30秒ごとにMyDashのデータと自動同期。「閉じる」ボタンでウィジェットを終了。

ウィンドウサイズ・位置の記憶

MyDashを閉じると、次回起動時に同じウィンドウサイズと位置で開く。複数モニター環境でも、最後に使っていたモニターに表示される。

💡 位置・サイズは10秒ごとに自動保存される。別のモニターに移動した場合も10秒待てば次回反映される。
⚠️ ウィンドウ位置の復元にはローカルサーバー（mydash-server.js）が起動している必要がある。MyDash.app から起動すれば自動で立ち上がる。

活用例：会議の資料を自動表示

🗓 会議前に資料・アプリを全部まとめて自動展開

セットとアラームを紐付けると、会議の時刻になるとポップアップで通知され、ワンクリックでセットが起動する。

準備ステップ

1
資料URLをブックマーク登録

例：Google スライド、Notion ページ、参考URL など

2
使うアプリをツール/アプリで登録

例：Zoom、Slack、Teams。Dockインポートでまとめて登録も可

3
「セット」を作成して必要なアイテムにチェック

例：「週次ミーティングセット」→ Google スライド + Zoom + Slack にチェック

4
セットの編集でアラームをONにして時刻を設定

例：毎週月曜 9:50（会議10分前）。これだけで完了！

🎯 アラームは MyDash が開いているタブでのみ発火する。MyDash.app を起動しっぱなしにしておくと確実。
